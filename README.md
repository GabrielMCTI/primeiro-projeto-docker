# 🚀 Docker Apache Login System

Aplicação web simples com fluxo de autenticação (login, cadastro e dashboard) rodando em container Docker com Apache.

---

## 📌 Tecnologias utilizadas

* 🐳 Docker
* 🌐 Apache
* 💻 HTML5
* 🎨 CSS3
* ⚙️ JavaScript

---

## 📂 Estrutura do projeto

```
.
├── docker-compose.yml
├── site/
│   ├── index.html
│   ├── cadastro.html
│   ├── dashboard.html
│   ├── style.css
│   └── script.js
```

---

## 🔐 Funcionalidades

* ✅ Tela de login
* ✅ Cadastro de usuário
* ✅ Dashboard com exibição do usuário
* ✅ Logout
* ✅ Controle simples usando localStorage

---

## ▶️ Como executar o projeto

### 1. Clone o repositório

```
git clone https://github.com/GabrielMCTI/meu-projeto-docker.git
```

### 2. Acesse a pasta

```
cd meu-projeto-docker
```

### 3. Suba o container

```
docker-compose up
```

---

## 🌍 Acesse no navegador

```
http://localhost:8080
```
